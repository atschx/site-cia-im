# cia.im 

Using Gatsby, hosted by Gatsby Cloud. 

## 🚀 Quick start

1. run with your laptop

    ```shell
    npm install
    ```

2. install gatsby-cli

    ```sh
    npm install -g gatsby-cli
    ```

3. just run it

    ```sh
    gatsby develop
    ```

4. click 

[🍺having fun！](http://localhost:8000/)
